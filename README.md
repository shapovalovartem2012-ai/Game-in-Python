# Game-in-Python

## Описания 
  RPG Игра созданая на Python. В игре есть анимаций персонажей,монеты,уровни.


## Тухнологии
  - Python
  - Pygame
  - JSON

##Установка и запуск
  - Клонируйте репозиторий
  - Создайте виртуальное окружение
  - Установите зависимости Pygame
  - Запустите сервер
